FocusTodo Pro Desktop v1.0.3

BUGFIX
1. 修复子任务文本框被通用 input 宽度样式挤压，导致无法输入的问题。
2. 子任务新增后自动聚焦，删除按钮改为事件绑定，不再依赖内联 onclick。
3. 任务卡片可直接勾选子任务并更新进度。

新增与补齐
1. 导入：XLSX / XLS / CSV / JSON。
2. 导出：XLSX / CSV（UTF-8 BOM）/ JSON。
3. 标准 XLSX 导入模板与填写说明。
4. 同 ID 导入更新、追加导入、替换导入、完整 JSON 备份恢复。
5. 文件附件选择、链接附件、打开附件。
6. 循环任务完成后自动生成下一次；支持跳过本次、终止循环。
7. 稍后提醒 10 分钟 / 1 小时，清单级提醒开关。
8. 月历 / 周历切换。
9. 关键词 + 清单 + 优先级 + 时间组合筛选。
10. 清单编辑、隐藏和恢复。
11. 默认清单和默认优先级设置。
12. 本周完成率与循环任务 30 天打卡日历。
13. 三级子任务视觉层级。
14. 基础加粗、列表、清单式备注。
15. 任务自定义颜色。
16. 任务可拖到左侧清单直接移动。
17. 系统托盘快速新建小窗口。

仍需后端或独立平台模块
1. 账号云同步与多端实时冲突合并。
2. 多人协作共享清单和权限系统。
3. 手机桌面小组件、手机震动、移动端后台能力。
4. 第三方日历 OAuth 双向同步。
5. 真正的好友在线分享与团队实时编辑。

以上功能不能仅靠单机 EXE 完整实现，需要云端服务、账号系统及对应移动端/日历平台授权。
